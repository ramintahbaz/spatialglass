# SpatialGlass

visionOS-style UI on iPhone using camera passthrough, gyro parallax, and knockout type that punches through to the camera feed.

Note: built for iPhone. Requires camera and motion permissions. Uses AVFoundation for passthrough and CoreMotion for parallax.
